# Form Fill Frontend

This folder contains the Fill workflow UI only.

- Page entry: `app/fill/page.tsx`
- UI component: `fill/FillClient.tsx`
- Backend API: `app/api/fill/chat/route.ts`
- AI insight API: `app/api/fill/insights/route.ts`
- Chatbot backend: `backend/fill/chatbot.ts`

Use this workflow for login-to-chat, spoken-style AI insight, and spreadsheet-style field filling.
