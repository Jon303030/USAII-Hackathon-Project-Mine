# Navigator Frontend

This folder contains the Navigate workflow UI only.

- Page entry: `app/navigate/page.tsx`
- UI component: `navigate/NavigateClient.tsx`
- Backend APIs: `app/api/navigate/questions/route.ts`, `app/api/navigate/search/route.ts`
- AI insight API: `app/api/navigate/insights/route.ts`
- Chatbot backend: `backend/navigate/chatbot.ts`

Use this workflow for report-query questions, keyword extraction, spoken-style AI insight, and search result selection.
