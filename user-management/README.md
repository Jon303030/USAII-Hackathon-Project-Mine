# User Management Frontend

This folder contains the User Management UI only.

- Page entry: `app/user-management/page.tsx`
- UI component: `user-management/UserManagementClient.tsx`
- Backend API: `app/api/users/route.ts`
- Roles: `user` and `admin`

Use this workflow to add demo users and review the current user list stored in PVC/local JSON.
